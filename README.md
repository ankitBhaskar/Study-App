# Study App

A React/Vite study MVP that turns an uploaded document into a mock study workflow.

## Current features

- Document upload screen with drag-and-drop interaction
- Generated study summary view
- Interactive quiz with scoring and weak-topic feedback
- Simulated 10-minute podcast player and transcript
- Tutor-style chat interface with mock AI response

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This version uses mock generated content only. The upload, summary, quiz, podcast and tutor behaviours are front-end demo interactions ready to be connected to a backend or AI layer.
